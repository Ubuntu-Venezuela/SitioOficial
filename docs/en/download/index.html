Version and system requirements

Ubuntu 24.04 LTS

Ubuntu 24.04 LTS, the latest LTS (Long term support) version provides 5 years of maintenance and security updates for free. In most cases, the LTS version is recommended for stable use.

  • See Ubuntu 24.04 LTS release notes

Ubuntu 24.10

If you want to try out the latest features, try out Ubuntu 24.10, the latest version. It provides 9 months of maintenance and security updates for free.

See Ubuntu 24.10 release notes

Ubuntu release and support cycle

A new version of Ubuntu is released every April and October with a 6-month interval. And a new LTS version is released in April of even years with a 2-year interval. Free maintenance and security updates are provided for 5 years for the LTS version and 9 months for interim releases. You may check out Ubuntu release cycle information on ubuntu.com

System requirements

  • 2 GHz dual-core processor and 4 GB system memory or better
  • 25 GB of free hard drive space
  • Internet access is helpful
  • Either a DVD drive or a USB port for the installer media
