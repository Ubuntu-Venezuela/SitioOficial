Sobre César Sevilla

César Sevilla es un notable contribuidor del entorno de Software Libre en Venezuela. Certificado globalmente como Miembro Oficial de Ubuntu, su trabajo constante de base fomenta la apropiación tecnológica en la región y facilita el ecosistema para nuevos usuarios que migran a sistemas de código abierto.
